# Local Music Shop — WordPress Block Theme

A dark, sophisticated WordPress block theme crafted for independent music shops. Built by [Miles AI](https://bymiles.ai), this theme brings the atmosphere of a late-night jam session to the web with warm tones, bold typography, and smooth interactive details.

## Overview

Local Music Shop is a full site editing (FSE) block theme designed for brick-and-mortar music retailers who want a strong online presence. It pairs a moody, high-contrast color palette with fluid typography and scroll-driven animations to create an experience that feels as alive as the stores it represents.

## Features

- **Full Site Editing** — Built entirely on WordPress block templates, template parts, and patterns for complete customization through the Site Editor.
- **Dark Aesthetic** — A carefully tuned palette of deep charcoals, warm caramels, and a punchy orange accent that evokes wood-paneled showrooms and stage lighting.
- **Bold Typography** — Big Shoulders Display for headlines and navigation paired with Crimson Pro serif for body text. All sizes scale fluidly across screen sizes.
- **Scroll Animations** — Elements fade and slide into view as visitors scroll, with staggered timing for a natural cascade effect.
- **Sticky Header with Blur** — A transparent header that gains a frosted-glass backdrop on scroll, keeping navigation accessible without obscuring content.
- **8 Page Templates** — Front page, blog index, single post, page, archive, search, 404, and a fallback index template.
- **7 Block Patterns** — Ready-made layouts for pages, posts, query loops, archives, search results, error pages, and comments.
- **Self-Hosted Fonts** — All typefaces are bundled as WOFF2 files for fast loading with zero external requests.
- **Mobile Ready** — Responsive layouts with an overlay mobile menu and touch-friendly interactions.

## Requirements

- WordPress 6.0 or higher
- PHP 7.4 or higher

## Installation

1. Upload the theme folder to `wp-content/themes/`.
2. Activate the theme from **Appearance > Themes**.
3. Open the **Site Editor** to customize templates, colors, typography, and layout to fit your shop.
